				Описание работы приложения:

Данное приложение представляет интернет магазин.

Возможности:
- добавление товаров в корзину
- оформление заказа
            
					Начало работы

1) Откройте проект решения Shop.sln
2) Cгенерируйте БД с клиентами. Для этого:
	а) В IDE Visual Studio откройте консоль менеджера пакетов NuGet.
	Путь: 'Tools' -> 'NuGet Package Manager' -> 'Package Manager Console'
	б) Введите команду Update-database, нажмите 'Enter'.
3) Запустите приложение. Ctrl + F5.
4) На главной странице нажмите кнопку "Choose file" и выберите файл Goods.xml в корневой папке с проектом.
5) Нажмите кнопку "Загрузить".

			Программные средства необходимые для запуска и работы

1) IDE Visual Studio версия .NET Framework 4.7.2
2) Библиотеки из NuGet. (Все необходимые библиотеки уже предустановлены в проект.)
